# Distributed Admission Handler

Distributed system class project.

## Usage

```sh
# Start a new server
python -m src --server

# Start a new monitor
python -m src --monitor

# Start a new cli client
python -m src --client

# Start a new ui client
python -m src --client --ui
```

## Troubleshoot

- Make sure your firewall is disabled
